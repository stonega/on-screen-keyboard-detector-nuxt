## Options

The fastest way to get started with **on-screen-keyboard-detector-nuxt** is to define the options:

```js {}[nuxt.config.js]
{
  modules: [
    ['on-screen-keyboard-detector-nuxt', {
      ...
    }]
  ],
}
```

*We strongly recommend to use environment variables for all these values!*
